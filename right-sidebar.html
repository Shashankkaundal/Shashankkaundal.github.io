Automation Project

Tweetbot in Pytest framework using python and selenium.

Purpose

The Purpose of tweetbot is to check and let other twitter users know if a specific product is avaialble on E-commerce website or not.

Functionality

The TweetBot program first checks on Amazon if Specifc product is available or not if avaialble the Bot will tweet the link of product and twitter users can land on product page via link and buy the product.

Tools & Technologies Used

The TweetBot is made using Selenium Automation Framework using Python in PyCharm IDE
